Estudio sobre bilingüismo

¡Muchas gracias por su interés en este proyecto! En este cuestionario se le mostrarán diferentes palabras en español y se le pedirá que indique la respuesta adecuada para cada una.

Una vez que acabe el cuestionario, pasará a formar parte de un sorteo en el que le puede tocar una tarjeta de regalo de 500 NOK para la tienda que elija.

Por favor, use un ordenador para completar el cuestionario (móviles y tabletas no están permitidos).
